# pica-minip

哔咔漫画 Minip 端 (目前只支持苹果)

**仅供个人学习使用**

## 安装

1. [App Store](https://apps.apple.com/us/app/minip-editor/id6463115915) 中下载 Minip App
2. 到 Release 界面手动下载最新版，并在 Minip App 首页右上角按钮中添加

   或使用 Minip App 扫描以下二维码:

   ![qrcode](./qrcode.png)
